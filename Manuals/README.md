# Manifold Manuals

This folder contains the **complete, player-facing manuscripts** for the Manifold Roleplaying Framework.

These PDFs are the authoritative texts for learning, running, and playing Manifold. They are written to be read directly and do not require reference to the underlying design specifications.

---

## Included Manuals

### **Manifold Game Master Manual**

The primary guide for running Manifold.

Covers how to:

* Facilitate play without controlling outcomes
* Apply world pressure and track consequence
* Use play modes, intent declaration, and dice correctly
* Exercise judgment consistently and transparently

**Required reading for GMs.**

* Safely expanding the world over time

## Notes

* These manuals assume **cooperative, good-faith play**.
* Manifold relies on **GM judgment**, supported by structure rather than replaced by rules.
* Dice are used sparingly to sample uncertainty, not to grant permission or determine success.

For system architecture, invariants, and implementation guidance, see the design documents in the root of the repository.
