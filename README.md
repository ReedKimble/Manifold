# Manifold Roleplaying Framework

Manifold is a **genre-agnostic tabletop roleplaying framework** designed around long-horizon consequence, shared reasoning, and cooperative judgment. It rejects traditional RPG assumptions (stats, classes, DCs, adversarial GMing) in favor of a **state-driven system** where possibility is defined by fiction and persistent state, and dice are used sparingly to sample uncertainty.

This repository contains the canonical manuscripts and design specifications that define Manifold.

---

## What Manifold Is

* A **shared cognitive system**, not a hidden rules engine
* A game about **consequence, pressure, and change**, not binary success/failure
* A framework that **requires and supports GM judgment** rather than replacing it
* Designed for **cooperative, good-faith play**

Manifold assumes that:

* The world exists independently of the players
* Pressure accumulates whether or not it is addressed
* Characters are shaped by history, not builds
* Dice never grant permission; they only sample uncertainty

## What Manifold Is Not

Manifold is deliberately **not**:

* A stat-based or class-based system
* A tactical combat engine
* A balance-focused or optimization-driven game
* A GM-vs-players experience
* A rules-complete engine that eliminates judgment

If you are looking for tight procedural resolution, build optimization, or adversarial challenge structures, Manifold is likely not a good fit.
